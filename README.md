## 스포츠 필드 SportsField

## 프로젝트 기획의도와 간략한 소개
스포츠는 혼자 할 수 없는 운동이고 여러명이 모여야 할 수 있는 운동이다.</br>
하지만 자신의 주변에서 나와 같은 운동을 하는 친구들은 찾기 힘들고 어려워</br>
나와 같은 운동 취미를 갖고 있는 사람들과 같이 운동 할 수 있도록 스포츠시설,</br>
날짜, 시간을 입력해 약속을 잡고 운동을 같이 할 수 있게 할 수 있는 App이다.
